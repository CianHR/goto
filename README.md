# goto
Command Line tool that allows you to set an active directory using `goto set` and easily return to it using `goto`

To install:

1. Clone the repo
2. you may need to make the install script executeable by chmod +x install.sh
4. Run the script by `./install.sh`
5. Open a new terminal
6. navigate to your currently most active directory and run `goto set` or use `goto set <path to directory>`
7. enjoy the ability to navigate to your project folder by running `goto` from anywhere
